# Ethereum Private Network ETH Transfer System

## Project Overview

A simple Web3.js-based ETH transfer system running on Ethereum Private Network.

## Features

- MetaMask Connection
- Account Address Display
- ETH Balance Check
- ETH Transfer
- Transaction Hash Verification

## Tech Stack

- Geth
- MetaMask
- Web3.js
- HTML
- CSS
- JavaScript
